() public

Defined in app/components/form-fields.js:376

This function if fired when a user releases a key on the keyborad. We are capturing this event in order to update the isTyping flag. We are taking the average typing speed of 41.4 words per minutes which is 0.69 words per second, which results in 1495ms per word.
