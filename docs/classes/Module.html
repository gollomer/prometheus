setupController

() private

Defined in app/controllers/app/module.js:98

Setup Controller function is called by the router at the end of the setupController function defined in the app.module route. This function is used to perform actions required at the start of each render cycle.

This is a replacement of init function which is only called once but if the setupController function in router calls this function then it is called every time a user tries to visit the route
